Get Started

Before using, we will explain you what is Typing.js. It's a library that allows to type your sentence(s) automatically when you make an animation in HTML.

Importation

To use Typing.js, Import via CDN or download it on Github
Normal file: 
https://cdn.jsdelivr.net/gh/Dob6458/typingjs/libs/typing.js (Normal file)
Minified file: 
https://cdn.jsdelivr.net/gh/Dob6458/typingjs/libs/typing.min.js (Minified file)
Then, import this file via <script> tag in your index.html file.

Initalization

Once imported, let's create a script.js file and make a link. 
<span id="typing"></script>
window.onload = function() {
  var typing = new Typing("#typing");
}
/!\: If you don't use window.onload, he doesn't work.
